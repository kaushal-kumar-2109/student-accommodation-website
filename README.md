# StayFinder - Student Accommodation Website

StayFinder is a responsive student accommodation website where users can explore PG properties, view details, filter by city/budget/gender, mark properties as interested, and send enquiries.

## Tech Stack

Frontend:
- React
- Bootstrap
- JavaScript
- Axios

Backend:
- PHP
- PDO

Database:
- Supabase PostgreSQL

## Features

- User signup and login
- Property listing
- Property filters
- Property details page
- Image gallery
- Amenities
- Interested / shortlist feature
- User shortlist page
- Contact owner enquiry form
- Admin enquiry view
- Responsive UI

## Run Frontend

```bash
cd frontend
npm install
npm run dev
